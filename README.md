# Trainee Grade Management System

This is a functional web application designed to manage student or trainee data. It allows users to add, view, and delete records while tracking performance.

## Features
- Add Trainees: Enter ID, Name, Department (DD, ID, PC), and Grade.
- Live Table Update: The list updates instantly when a trainee is added or removed.
- Automatic Calculation: The system identifies and displays the highest grade in the list.
- Data Validation: Simple alerts to ensure all fields are filled before submitting.

## Built With
- HTML5
- CSS3 (Clean UI design)
- JavaScript (Array manipulation and DOM logic)

## Purpose
I built this project to practice JavaScript fundamentals, specifically working with objects, arrays, and dynamic HTML updates. It is part of my technical training in the Digital Development program.
